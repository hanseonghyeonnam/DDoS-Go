# DDoS-Go, dGo
The DDoS tool for Go language.

## Usage
1. Build the project.
   ```shell
   go build .
   ```
2. Run!
 >  ./dgo [options]

## Options
  - -url, set the url that will attacked.
  - -DPS, set the DDoS-Per-Second, DPS
